# Water-Management-System

## Overview

This project simulates water consumption and billing for an apartment based on:

* Apartment type
* Water source ratios (Corporation vs Borewell)
* Additional guests

## Project Structure

### 1. interview-solution

The original implementation written during a coding interview.

* Focused on solving the problem quickly
* Contains tightly coupled logic and global state
* Represents real-time problem-solving under constraints

### 2. refactored-solution

An improved version of the same solution using clean code principles.

* Modular design with separation of concerns
* Improved readability and maintainability
* Structured into handler and logic layers

## Key Learning

This project demonstrates the transition from:

* Writing code for correctness → Designing code for clarity and scalability

## How to Run

1. Compile the files:
   javac Main.java

2. Run:
   java Main
